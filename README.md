# Space-X Task Creator!


## Installation
### `git clone https://github.com/AgusPk/Space-X.git`

Copy .env.example

### `cp .env.example .env`

Fill the variables TRELLO_KEY and TRELLO_TOKEN with your trello account credentials

Install dependencies
### `npm install`

Bring up the server
## `npm start`
